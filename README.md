# 프레시-이지

## 프로젝트 요약

기존의 프레시코드 사이트 클론 프로젝트

기존 사이트에서 사용자 입장에서 편리하게 사이트를 이용할 수 있도록 개선하기

주력 상품인 샐러드 구독을 강조하고, 상품 나열 방식이 아닌 일관된 user-flow를 부연하여 구독 과정을 쉽게 만듦

---
